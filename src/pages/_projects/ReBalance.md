Eu desenvolvi o ReBalance para facilitar o rebalanceamento de uma carteira de investimentos. O primeiro protótipo do aplicativo foi criado em [SolidJS](https://www.solidjs.com), mas conforme fui acrescentando mais funcionalidades decidi migrar o projeto para o [Remix](https://remix.run), que é um framework muito mais maduro. Até hoje, esse é meu projeto favorito pela sua utilidade.
